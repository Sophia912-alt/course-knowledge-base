## 第一步：下载这个程序

### 🖥 Mac 用户

1. 打开浏览器，访问这个网址：
   ```
   https://github.com/Sophia912-alt/course-knowledge-base
   ```

2. 在页面上找到绿色的 **「Code」** 按钮，点击它

3. 在弹出的菜单里，点击 **「Download ZIP」**

4. 下载完成后，双击这个 ZIP 文件，它会自动解压成一个文件夹
   - 文件夹名字大概是 `course-knowledge-base-main`

5. 把这个文件夹**移到桌面**上，这样好找
   - 你可以直接把它拖到桌面上

> 💡 小提示：如果你看到文件夹名字后面有 `-main`，可以重命名把 `-main` 删掉，变成 `course-knowledge-base`，但这不是必须的。

### 🪟 Windows 用户

1. 打开浏览器，访问这个网址：
   ```
   https://github.com/Sophia912-alt/course-knowledge-base
   ```

2. 在页面上找到绿色的 **「Code」** 按钮，点击它

3. 在弹出的菜单里，点击 **「Download ZIP」**

4. 下载完成后，右键点击这个 ZIP 文件 → **「全部解压缩」**
   - 解压后的文件夹名字大概是 `course-knowledge-base-main`

5. 把这个文件夹**移到桌面**上

## 第二步：安装 Python（运行程序需要的东西）

### 🖥 Mac 用户

1. 打开浏览器，访问：
   ```
   https://www.python.org/downloads/
   ```

2. 你会看到一个黄色的大按钮，上面写着 **「Download Python 3.xx.x」**
   - 点击它！它会下载一个安装包

3. 下载完成后，双击这个安装包（文件名是 `python-3.xx.x-macos11.pkg` 这样的）

4. 一路点 **「继续」** → **「安装」** → 输入你的电脑密码 → **「安装软件」**

5. 安装完成后，点 **「关闭」**

> ✅ 装好了！就这么简单。

### 🪟 Windows 用户

1. 打开浏览器，访问：
   ```
   https://www.python.org/downloads/
   ```

2. 你会看到一个黄色的大按钮，上面写着 **「Download Python 3.xx.x」**
   - 点击它！

3. 下载完成后，双击这个安装包

4. **⚠️ 这一步非常重要！**
   - 在安装窗口的**最下面**，有一个选项叫 **「Add Python to PATH」**
   - **一定要勾选它！**（点一下那个小方框，出现 ✅）
   - 然后点 **「Install Now」**

5. 等它安装完成，点 **「Close」**

> ✅ 装好了！

## 第三步：安装依赖（补一个零件）

这个程序需要一个叫 `requests` 的小零件才能正常工作。我们需要在终端里装一下。

### 🖥 Mac 用户 — 打开终端

1. 在电脑右上角点击 **放大镜图标**（Spotlight 搜索）
2. 输入 `终端` 或 `Terminal`
3. 点击 **「终端」** 打开它（会弹出一个黑乎乎的窗口）

### 🪟 Windows 用户 — 打开命令提示符

1. 点击屏幕左下角的 **开始菜单**（Windows 图标）
2. 输入 `cmd`
3. 点击 **「命令提示符」** 打开它（会弹出一个黑乎乎的窗口）

### 在终端里输入命令

1. 在刚才打开的黑色窗口里，**复制**下面这行命令（选中后按 Ctrl+C）：
   ```
   pip3 install requests
   ```

2. 在黑色窗口里**粘贴**（Mac 按 Command+V，Windows 按 Ctrl+V），然后按 **回车键（Enter）**

3. 你会看到一些文字刷刷地滚动，等它停下来

4. 当它停下来后，最后一行应该是类似这样的：
   ```
   Successfully installed requests-2.xx.x
   ```

> ✅ 装好了！你可以关掉这个黑色窗口了。

## 第四步：配置 API Key（让 AI 帮你分析）

这个程序需要用一个 AI 来分析你的课程内容。你需要一个密钥（API Key）来使用 AI。

### 如果你有 DeepSeek 的 API Key

1. 在桌面上找到你刚才下载的文件夹，双击打开

2. 找到 **`run_pipeline_with_key.sh`** 这个文件

3. **右键点击**它 → **「打开方式」 → 「文本编辑」**（Mac）或 **「记事本」**（Windows）
   - ⚠️ 不要双击运行！要用文本编辑器打开

4. 你会看到文件里有一行这样的文字：
   ```
   export OPENAI_API_KEY="YOUR_API_KEY_HERE"
   ```

5. 把 `YOUR_API_KEY_HERE` 替换成你真实的 API Key
   - 比如你的 Key 是 `sk-abc123`，就改成：
   ```
   export OPENAI_API_KEY="sk-abc123"
   ```

6. **保存文件**（Mac 按 Command+S，Windows 按 Ctrl+S），然后关掉

### 如果你还没有 API Key

别担心，去申请一个，很简单：

1. 打开浏览器，访问 https://platform.deepseek.com/
2. 注册一个账号（用邮箱就行）
3. 登录后，点左边菜单的 **「API Keys」**
4. 点 **「创建 API Key」**
5. 复制这个 Key，然后按上面的步骤填进文件里

> 💡 第一次注册 DeepSeek 会送一些免费额度，够你用好久了。

## 第五步：放入你的课件和转写

现在把你要整理的课程资料放进去：

### 1. 放课件

1. 打开桌面上的文件夹，找到 **`raw-slides/`** 这个文件夹
2. 把你的课件（.md 文件）**拖进去**

> 课件就是老师发的课程笔记，一般是 Markdown 格式（文件名以 `.md` 结尾）

### 2. 放转写

1. 找到 **`raw-transcripts/`** 这个文件夹
2. 把录音转写的文本（.txt 文件）**拖进去**

> 转写就是用软件把课堂录音转成文字的文档（文件名以 `.txt` 结尾）

### 文件夹结构示例

```
course-knowledge-base/           ← 你下载的文件夹
├── raw-slides/                  ← 📥 把课件拖到这里
│   ├── 第一课 认识AI.md
│   └── 第二课 提示词技巧.md
│
├── raw-transcripts/             ← 📥 把转写拖到这里
│   ├── 第一课 录音转写.txt
│   └── 第二课 录音转写.txt
│
├── run_pipeline_with_key.sh     ← ⚙️ 等下运行这个
└── ...其他文件
```

## 第六步：运行程序

### 🖥 Mac 用户

1. 在文件夹里找到 **`run_pipeline_with_key.sh`**
2. **右键点击**它 → **「打开方式」 → 「终端」**
   - ⚠️ 第一次运行可能会弹出一个警告，说「无法验证开发者」
   - 点 **「仍要打开」**
3. 一个黑色窗口会打开，里面有很多文字在跑
4. **等着就行**，大约需要 3-5 分钟
   - 如果一切顺利，最后你会看到：
   ```
   ✅ 导出完成
   ```
5. 窗口会自动关闭，或者你可以手动关掉

> 💡 如果双击后闪一下就没了，试试右键 → 打开方式 → 终端

### 🪟 Windows 用户

Windows 用户需要先做一个小调整：

1. 在文件夹里找到 **`run_pipeline_with_key.sh`**
2. **右键点击**它 → **「重命名」**
3. 把文件名改成：**`run_pipeline.bat`**
   - ⚠️ 如果系统提示「更改扩展名可能导致文件不可用」，点 **「是」**

4. **右键点击**改好后的 `run_pipeline.bat` → **「以管理员身份运行」**

5. 一个黑色窗口会打开，里面有很多文字在跑
6. **等着就行**，大约需要 3-5 分钟
7. 看到 `✅ 导出完成` 就表示搞定了

## 第七步：在 Obsidian 里看结果

现在你可以用 Obsidian 来阅读自动生成的课程笔记了。

### 安装 Obsidian（如果还没装）

1. 打开浏览器，访问 https://obsidian.md/
2. 点击 **「Download」** → 选择你的系统版本下载
3. 安装它（和装其他软件一样）

### 打开知识库

1. 打开 Obsidian
2. 你会看到一个窗口，左边有一个 **「打开」** 按钮（或者叫 **「Open folder as vault」**）
3. 点击它 → 找到你桌面上的 `course-knowledge-base` 文件夹 → **「选择」**

4. 你会看到 Obsidian 左边出现一个文件列表：

```
🏠 知识库总览.md     ← 点这个！这是入口
📋 知识库条目.md     ← 所有知识点的汇总
📚 课程总结/         ← 每一节课的笔记
📦 资产库/           ← 工具/网站/技能清单
```

### 怎么看

1. 点击 **「🏠 知识库总览.md」** — 这是整个知识库的首页
   - 上半部分列出了所有课程，**点击课程名字**就能跳到对应笔记
   - 下半部分有工具和技能的快捷入口

2. 如果想找某个工具：
   - 点击左边 **「📦 资产库」** 前面的小箭头展开
   - 点击 **「🛠 工具清单.md」** — 所有工具按出现的次数排列

3. 如果想回顾某节课：
   - 点击左边 **「📚 课程总结」** 展开
   - 点你想看的那节课

## 每次学完新课后怎么办

以后每次上完新课，只需要三步：

### 第一步：放文件
把课件拖进 `raw-slides/`
把转写拖进 `raw-transcripts/`

### 第二步：运行
双击 `run_pipeline_with_key.sh`（Mac）或 `run_pipeline.bat`（Windows）

### 第三步：看结果
回到 Obsidian，刷新一下（点右上角的刷新按钮，或者直接关闭重新打开）

> 💡 **整个过程不到 1 分钟**，之后你就能在 Obsidian 里看到新课程的笔记了。

## 遇到问题怎么办

### ❓ 双击脚本后没反应
- **Mac**: 右键 → 打开方式 → 终端
- **Windows**: 右键 → 以管理员身份运行

### ❓ 黑色窗口一闪就没了
说明程序出错了。可以这样看错误信息：
- **Mac**: 打开终端 → 输入 `cd Desktop/course-knowledge-base` 按回车 → 输入 `bash run_pipeline_with_key.sh` 按回车 → 看屏幕上显示的错误信息
- **Windows**: 同上，打开命令提示符，输入 `cd Desktop\course-knowledge-base` → `run_pipeline.bat`

### ❓ Python 安装报错
- 可能是下载错了版本，重新去 python.org 下载一次
- 或者请身边懂电脑的朋友帮看一下

### ❓ 出现「ModuleNotFoundError: No module named 'requests'」
- 说明第三步没装好。重新执行第三步：
  - 打开终端/命令提示符
  - 输入 `pip3 install requests`
  - 按回车，等它装完

### ❓ 运行后说「API Key 未设置」
- 你没有正确填写 API Key
- 重新打开 `run_pipeline_with_key.sh`，检查 `YOUR_API_KEY_HERE` 有没有被替换成你的真实 Key

### ❓ Obsidian 里看不到新课程
- 运行一次程序，然后刷新 Obsidian（点右上角的刷新图标）
- 或者关掉 Obsidian 重新打开
